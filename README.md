<h1 align="center">💈BARBER SHOP</h1>

## 📝 Descrição
 Este projeto é uma landing page com tema de barbearia feito com o objetivo de aprender e desenvolver técnicas e utilizar funcionalidades de SASS
---
## 🛠 Tecnologias
- HTML
- CSS
- SCSS
---

## 📂 Estrutura do Projeto

```bash
.
├── assets/
│   └── imagens/        # Arquivos de imagem do projeto
├── partials/           # Arquivos SCSS modulares
│   ├── colors.scss     # Variáveis de cores
│   ├── header.scss     # Estilo do header
│   └── layout.scss     # Layout geral
├── index.html          # Página principal
├── style.scss          # Arquivo SCSS principal
├── style.css           # CSS compilado
└── style.css.map       # Source map
```
